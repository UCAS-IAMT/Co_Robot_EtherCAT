# Elmo
使用软件之前，要先插上通讯线，连接电机驱动板和电脑的usb

**机械臂只要上电即可，不需要启动ubuntu里面的ethercat通讯，否则会出现抢断**

## 1. 电机整定

1-1  上电 & 通讯电机
